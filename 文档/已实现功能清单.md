## 十六、快速上手

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问地址
http://localhost:5173
```

### 典型工作流

1. **导入模型** — 点击顶部「导入模型」，或拖拽 GLB/GLTF 到视口
2. **编辑对象** — 按 W/E/R 切换移动/旋转/缩放，或在右侧属性面板输入精确数值
3. **调整材质** — 选中对象 → 右侧「材质」Tab → 调节颜色、金属度、贴图等
4. **布置灯光** — 左下组件库 → 添加灯光 → 右侧调节参数
5. **全局设置** — 取消选择 → 右侧全局设置 → 背景、HDR、后期效果等
6. **导出** — 点击顶部「导出」→ 选择 GLB / 截图 / JSON 配置
